Get-Menseki
====

## これは何

WingNeo の画地一覧をコピペしたテキストファイルから面積の一覧を作成します。  
面積はフル桁の状態から、小数第3位以下を切り捨てた面積となります。

## 使い方

メモ帳で`Get-Menseki.ps1`を開き、下記の3項目を設定してください。  
画地一覧の画面で`大字`、`区画名称`、`座標面積`がそれぞれ何列目にあるかを見て書き換えます。  
画面に表示されていない場合には`表示切り替え`のボタンから表示項目を追加しておく必要があります。

```
#大字のカラム番号
$o = 1
#区画名称のカラム番号
$k = 2
#座標面積のカラム番号
$z= 6
```

画地一覧の画面で、面積を取り出したい画地を選択し、`No.`のカラムで右クリック→`コピー(行単位)`を選択します。  
スタートメニューからメモ帳を開き、クリップボードの内容を貼り付けて適当な名前を付けて保存します。

下記の様に実行すると`menseki.txt`と同じフォルダに`menseki.txt.arrange.csv`が作成されます。

```
PS1> Get-Menseki.ps1 menseki.txt
```

`menseki.txt.arrange.csv`を Excel で開くとA列に所在、B列に地番、C列に面積が入った状態になっているはずです。
